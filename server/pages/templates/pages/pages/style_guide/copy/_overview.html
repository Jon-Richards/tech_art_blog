Copy

This page contains usage examples for styling general content.

The styles below use type selectors (HTML tags) to keep specificity low and provide leeway for the content to style itself. Content written with bare html tags simply needs to be placed in a tag with a class of "copy" for these styles to be applied. 

        
          <div class="copy">
            <h1>Hello World!</h1>
            <p>
              The h1 and this p tag will have copy styles.
            </p>
          </div>

Styles can also be applied in-line like so: 

        
          <p class="copy">
            This paragraph, and only this paragraph, will have copy styling.
          </p>

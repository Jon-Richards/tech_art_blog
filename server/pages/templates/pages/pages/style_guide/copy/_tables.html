There also exists a custom element for a table. The custom element is (slightly) less verbose, implements its own wrapper element, and is purely data-driven. 

        
          <jr-table
            data-caption="This is a caption."
            data-head='["Column 1", "Column 2", "Column 3"]'
            data-rows='[
              {
                "hasHeading": true,
                "cells": ["This is cell 1", "This is cell 2", "This is cell 3"]
              },
              {
                "cells": ["This is cell 1", "This is cell 2", "This is cell 3"]
              }
            ]'
          ></jr-table>

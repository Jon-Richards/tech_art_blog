Links

I am a link.

Links and anchor elements are designed specifically for navigation. They should not be intermixed with buttons, which have entirely different semantic meaning. 

        
          <div class="copy">
            <a href="#">I am a link!</a>
          </div>

          <a href="#" class="copy-p">I am a link!</a>

CSS Custom Property API

        
          .copy a:link, .copy-a:link {
            --color: green;
          }

Links & Anchors

Links

I am a link.

Links and anchor elements are designated specifically for navigation. Links should not mixed with buttons, which serve a different purpose from a semantic standpoint.

Good

This is an interesting YouTube video entitled The Art of Code by Dylan Beattie.

  • The link has a meaningful text label.
  • The link cannot be disabled.

Bad

Open Modal

Links are not the same as buttons, which perform UI operations, typically without losing context, e.g. submitting a form or opening a modal.

Additional Conventions
  • External links should always open in a new tab.
